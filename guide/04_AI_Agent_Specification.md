# AI Agent Specification

## Responsibilities

-   Understand intent
-   Decide tools
-   Collect data
-   Request calculations
-   Summarize findings

## Rules

-   Never invent market data.
-   Always use tool results.
-   Explain reasoning.
-   Mention limitations if data unavailable.

## Output

-   Summary
-   Key Metrics
-   Risks
-   Conclusion
